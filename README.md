🧩VISÃO DO DIA A DIA
Como é o trabalho de um(a) Analista de Dados:
• Coletar dados de diversas fontes (bancos de dados, planilhas, APIs).
• Limpar e preparar dados para análise.
• Criar dashboards e relatórios visuais (Power BI, Tableau).
• Analisar métricas e identificar insights acionáveis.
• Ajudar times de negócio a tomar decisões baseadas em dados.
🧠MAPA DE SKILLS
CORE SKILLS (essenciais):
• Lógica e pensamento analítico
• SQL para consultas e manipulação de dados
• Python básico para análise e automação
NICE-TO-HAVE (complementares):
• Estatística descritiva
• Storytelling com dados
FERRAMENTAS E TECNOLOGIAS:
• Power BI
• Tableau
• Python (Pandas, Matplotlib)
📅ROADMAP DE 90 DIAS
ADAPTADO PARA: 10h/semana
MÊS 1 – FUNDAMENTOS
SEMANA 1-2:
• Aprender fundamentos de dados (tipo de dados, ciclo analítico).
• Instalar e configurar ambiente Python + Jupyter Notebook.
SEMANA 3-4:
• SQL básico: SELECT, WHERE, ORDER BY, LIMIT.
• Python básico: variáveis, listas, dicionários, loops.
MÊS 2 – PRÁTICA
SEMANA 5-6:
• SQL intermediário: JOINs, GROUP BY, HAVING.
• Power BI: conectar dados, transformar, montar visualizações simples.
SEMANA 7-8:
• Python para análise de dados: Pandas (importar, limpar, transformar).
• Criar gráficos com Matplotlib/Seaborn.
MÊS 3 – PORTFÓLIO E PREPARAÇÃO
SEMANA 9-10:
• Criar dashboard completo no Power BI ou Tableau.
• Analisar dataset público e estruturar insights como casos de negócio.
SEMANA 11-12:
• Montar portfólio no GitHub + LinkedIn.
• Treinar roteiro de entrevistas técnicas e comportamentais.
🚀PROJETO DE PORTFÓLIO
PROJETO: Dashboard de Performance de Jogos – Análise de Players
O QUE FAZER:
Monte um dashboard completo usando dados de jogos (ex: Steam, League of Legends, FIFA
ou dataset do Kaggle).
Seu objetivo é mostrar métricas como: engajamento, tempo jogado, personagens favoritos,
taxas de vitória.
ENTREGÁVEIS:
• Dataset tratado (SQL/Python).
• Dashboard interativo no Power BI ou Tableau.
• Relatório de insights com 5 a 8 descobertas.
CRITÉRIOS DE ACEITAÇÃO:
• Dashboard com pelo menos 6 visualizações.
• Filtros interativos funcionando.
• Storytelling final respondendo “o que isso significa?”.
DICA:
Use cores e ícones que remetam a jogos — isso enriquece o design e reforça sua
personalidade como analista.

COMO RESPONDER:

PERGUNTA 4: "Como você transforma dados em insights?"
COMO RESPONDER:

PERGUNTA 5: "Conte um projeto que você tem orgulho."
COMO RESPONDER:
Descreva seu projeto de jogos: problema, abordagem, ferramentas e resultados.
